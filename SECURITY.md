# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We welcome vulnerabilty reports. Please
[open an issue](https://github.com/oderayi/map-transform-cjs/issues/new) and
describe what you've find.
